# Margdarshak Demo

Minimal React + Vite + TypeScript + Tailwind app running a Leaflet map and lightweight UI primitives.

## Run locally (Windows PowerShell)

1. Install dependencies
```
npm install
```

2. Start dev server
```
npm run dev
```

3. Open the printed local URL (usually http://localhost:5173).

## Build
```
npm run build
npm run preview
```
